## How To Use

In order for the application to work, you must have a valid [Mon Bureau NumÃ©rique](https://www.monbureaunumerique.fr/) account, [git](https://git-scm.com/), and you have to get tokens with [scolengo token](https://github.com/maelgangloff/scolengo-token). From your command line:

```bash
# Clone this repository
$ git clone https://github.com/damienRifflart/StudyStats

# Go into the repository
$ cd StudyStats

# Install dependencies
$ npm install

# Create config.js file src/main/ and write:
config = (your json from scolengo token)

# Run the app
$ npm run dev
```
